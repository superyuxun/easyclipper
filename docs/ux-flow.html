No account. No internet round-trips. No cost. Your audio never leaves the machine — the only thing ever downloaded is the speech-recognition AI, once.

1

Drop in your audio

Drag any audio (or video) file onto the upload screen. EasyClipper fingerprints it so that if you ever open the same file again, it skips straight to the transcript.

2

It transcribes itself

The Whisper AI runs inside the browser and writes out every spoken word with an exact timestamp. The first run downloads the AI model (~150 MB, a few minutes). After that, it's instant.

3

Cut by clicking words

Click one word to start a highlight, click another to end it. The audio between them is pulled out instantly — no waveform scrubbing.

4

Polish the clip

Open the Detail View to fine-trim the ends with a slider, or speed up part of the clip (1.5×–4×) to tighten a slow patch. The preview updates as you go.

  • Trim slider for exact in/out points
  • Speed edits on any sub-section
  • Live preview player
